# TCP vs UDP

As part of my cybersecurity learning journey (Day 4), I explored the difference between TCP and UDP.

## 🔹 What is TCP?
TCP (Transmission Control Protocol) is a connection-oriented protocol.

### Features:
- Reliable communication
- Data is sent in order
- Error checking and retransmission

### Example Use Cases:
- Websites (HTTP/HTTPS)
- Email
- File transfer

---

## 🔹 What is UDP?
UDP (User Datagram Protocol) is a connectionless protocol.

### Features:
- Fast communication
- No guarantee of delivery
- No error checking

### Example Use Cases:
- Online gaming
- Video streaming
- VoIP calls

---

## 🔥 Key Difference

| Feature        | TCP              | UDP            |
|----------------|------------------|----------------|
| Speed          | Slower           | Faster         |
| Reliability    | High             | Low            |
| Connection     | Connection-based | Connectionless |
| Use Case       | Secure transfer  | Real-time apps |

---

## 💡 My Understanding

TCP focuses on accuracy and reliability, while UDP focuses on speed and performance.
