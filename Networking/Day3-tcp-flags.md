# Day 3 – TCP Flags 🚀

## 🔹 What are TCP Flags?
TCP flags are control bits used to manage and control TCP connections.

They help in:
- Starting a connection  
- Managing data transfer  
- Closing a connection  

---

## 🔹 Important TCP Flags

### 🔥 SYN (Synchronize)
Used to initiate a connection.

### 🔥 ACK (Acknowledgement)
Confirms that data is received.

### 🔥 FIN (Finish)
Used to close a connection gracefully.

### 🔥 RST (Reset)
Immediately terminates a connection.

### 🔥 PSH (Push)
Forces data to be sent immediately.

### 🔥 URG (Urgent)
Indicates urgent data.

---

## 🔹 3-Way Handshake

1. SYN → Client sends request  
2. SYN-ACK → Server responds  
3. ACK → Client confirms  

👉 Connection established successfully.

---

## 🔹 Cybersecurity Importance

- Helps detect abnormal network behavior  
- Used in attack detection (SYN Flood)  
- Important for SOC analysis and monitoring
